git命令行(commit流程)
    ↓	git branch							 ---查看本地分支
    ↓	git pull							---同步分支
    ↓	git add .							---暂存修改
    ↓	git commit -m "提交修改说明"		 ---提交
    ↓	git checkout -b xxxxxx				---新建并切换分支
    ↓	git push origin xxxxxx				---上传分支
    ↓	分支推送到远程之后，在GitHub上找到它，然后New/Create Pull request
    ↓	Merge pull request
